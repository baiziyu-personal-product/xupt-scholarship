# xupt-scholarship

XUPT研究生学业奖学金评定系统设计与实现

## 项目介绍

采用`Go`语言开发，使用`Iris`框架，数据库采用`Redis`+`Mysql`。

使用`Viper`读取配置文件

## 项目特点

1. 支持使用`dchest/captcha`创建图片验证码

## 项目接口

所有的项目接口同步在`Postman`
